Project : Progress Bar Assignment
Developer : Poonam Meghnani
Date : 15-Feb-2017
Entry Point : index.html

Description:
The project reads configuration from an endpoint and creates progress bars as per the endpoint output.

For example: If endpoint result is : {"buttons":[20,40,-32,-44],"bars":[33,22,83],"limit":170}, you will see -
	- four buttons that adds percentage to progress bar as per their value the output.
	- three progress bars with max limit 170
	
To run the project, open index.html in any browser.
